# 介绍

起点是北大大佬的网站[CS自学指南](https://csdiy.wiki/)，里面写的当然比我的好，内容也当然比我的难，没有基础去看的话可能不知如何下手。

所以这个项目的定位是**纯萌新**与CS自学指南之间的一层缓冲，帮助初学门理解计算机专业的大致框架，减少刚入门时的阻力。
